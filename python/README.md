RPG Combat Approval Kata - Python version
=========================================

There are several implementations of 'rpg_combat' in this folder. There is a single approval test available in 'test_rpg_combat'. Write approval tests for the code in 'rpg_combat' then use these tests to identify the bugs in the other versions called 'rpg_combat_bug1', 'rpg_combat_bug2' etc.

Development Environment
-----------------------

Suggestion: create a python virtual environment for this project. See the [documentation](https://docs.python.org/3/library/venv.html)

install requirements:

    pip install --upgrade -r requirements.txt

run tests:

    python -m pytest
